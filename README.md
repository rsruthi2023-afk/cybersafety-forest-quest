# Cyber Safety Forest Quest

## Project Documentation
The Cyber Safety Forest Quest is an interactive and educational game designed to teach players about online safety and cybersecurity concepts. Players navigate through a virtual forest, facing various challenges and scenarios related to internet safety.

---

## Features
- **Interactive Gameplay**: Players encounter different challenges that require decisions to ensure their safety online.
- **Educational Content**: Each challenge is designed to impart knowledge about cybersecurity practices in a fun and engaging manner.
- **User-friendly Interface**: The game features an easy-to-navigate interface suitable for players of all ages.
- **Progress Tracking**: Players can track their progress and learn from their decisions during the game.

---

## Installation Guide
1. **Clone the Repository**: Clone this repository to your local machine using the following command:
   ```bash
   git clone https://github.com/rsruthi2023-afk/cybersafety-forest-quest.git
   ```

2. **Navigate to Project Directory**: Change your directory to the project folder:
   ```bash
   cd cybersafety-forest-quest
   ```

3. **Install Dependencies**: Install the necessary dependencies using:
   ```bash
   npm install
   ```

4. **Run the Game**: Start the game with:
   ```bash
   npm start
   ```

---

## Gameplay Instructions
- **Starting the Game**: Launch the game from your terminal after following the installation steps.
- **Making Decisions**: As you navigate through the forest, you will be faced with various scenarios. Make decisions that reflect good cybersecurity practices.
- **Learning Points**: After each scenario, you will receive information and tips related to online safety based on your choices.
- **Complete the Quest**: Aim to complete the game with the best outcomes in maintaining online safety.
